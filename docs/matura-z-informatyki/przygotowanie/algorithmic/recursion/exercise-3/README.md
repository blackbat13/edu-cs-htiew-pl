# Ćwiczenie 3

Zapoznaj się z poniższą specyfikacją oraz pseudokodem, a następnie rozwiąż zadania.

## Specyfikacja

### Dane

* n - liczba naturalna, $n>0$. 

## Pseudokod

```
funkcja fun(n):
    1. Jeżeli n = 0, to
        2. Zakończ
    3. Jeżeli n mod 2 = 0, to
        4. Wypisz "0"
        5. Wywołaj fun(n div 2)
    6. Jeżeli n mod 2 = 1, to
        7. Wypisz "1"
        8. Wywołaj fun(n div 2)
```
